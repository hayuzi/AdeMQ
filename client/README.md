交互式命令行客户端
==

### 1. 目前实现以下基础功能
- 交互式命令客户端的命令读取
- 命令解析器（负责解析命令并格式化）
- 命令分发器（负责根据命令路由分发命令到处理函数）
- 命令处理器（负责根据具体参数处理数据）
- 远程服务连接 （连接到远程服务器（单连接方式实现））
- - 服务器数据收发
- - 独立协程负责从命令处理器接收命令并发送到服务器
- - 独立协程负责从服务器接受结果并发还到命令处理器
- - 独立协程发送心跳包维持与服务器的连接

### 2. 使用示例

启动客户端:
```
# 连接到给定地址的远程服务
go run client.go -address=127.0.0.1:10601
```

启动客户端之后执行命令:
```
help    命令查看帮助信息
history 查看历史记录
ping    向远程服务器发送基本消息
```
